** How to debug use cmake
press C+F5
** How to pass args to cmake debug
 I was able to pass arguments to a debug target using cmake.debugConfig adding a settings.json file inside .vscode folder with this content:

 #+begin_src 
{ 
    "cmake.debugConfig": {
        "args": [
            "myFirstArgument"
        ]
    }
}
 #+end_src
